Display a splash screen.

== Description ==

Display a splash screen to give information to your visitor before they get to the wordpress blog.

The administrator can customize the splash screen and parameters for the cookie to remember visitors.

== Installation ==

Extract the zip file content after download and copy the directory "splashscreen" to the plugin directory located at "wp-content/plugins". Login as admin and enable "Splashscreen" through plugin tab.
